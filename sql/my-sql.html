MySQL

在 Mac 上搭建MySQL数据库

  1. Download MySQL Community Server
  2. 下载dmg格式的文件安装，最后一步会给root用户生成一个临时密码，截图或者复制密码等下登陆用
  3. 安装完成之后，打开系统偏好设置，最下边一栏有个MySQL图标，点击之后选择Start MySQL Server运行数据库
  4. 打开终端Terminal，可以从Command+空格（Space）跳出Spotlight搜索栏中打开
  5. 输入指令，进入到bin文件夹下，cd指令为进入用户主目录，后面为文件路径，跟手动打开目录是一个意思 cd /usr/local/mysql/bin
  6. 这条指令让我们输入主机名、用户名、密码，我们是本地登陆可以不输主机名，用root账户，密码为刚刚保存的默认密码 ./mysql -h host -u username -p
  7. 完整指令./mysql -h localhost -u root -p
  8. 然后在执行其他操作之前，数据库会要求我们修改默认密码SET PASSWORD FOR 'root'@'localhost' = PASSWORD ('new-password-here');
  9. 在括号里填root用户的密码root或者记得住的其他密码，然后重新执行第二步登陆数据库，输入密码
  10. 输入下面这条指令，确保root账户的密码不会过期，不推荐在实际操作的数据库中中使用ALTER USER 'root'@'localhost' PASSWORD EXPIRE NEVER;
  11. 输入指令确认数据库能正常使用然后退出use mysql
  12. 官网有提供对不同情况重置密码的方式Reference Manual
  13. 或者可以参考stackoverflow上的解决方案Stack Overflow
